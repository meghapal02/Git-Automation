# Git Project Initialization Automation (Linux OS)

## Command:

### For making `public repo`:
```bash
  $ create.py -n <repo name>
   or
  $ create.py --name <repo name> 
```   

### For making `private repo`:
```bash
  $ create.py -n <repo name> -p
   or
  $ create.py --name <repo name> --private 
```
